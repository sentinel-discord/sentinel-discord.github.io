Sentinel Bot

Randomized League of Legends teams and builds.

About

Sentinel Bot is a Discord bot that randomly creates teams of users from a voice channel and generates League of Legends builds. Supported game modes include Summoner's Rift (Ultimate Bravery) and ARAM.

Users can effortlessly set up scrims and generate builds without ever having to leave the Discord application!

Add Bot!

Commands

Commands Description
!r help Provides a list of available commands when using Sentinel Bot.
!r teams Creates a new poll where you can select users to be randomized.
!r same Randomizes the same list of users that was chosen previously.
!r sr [same] Generates random champions and builds for Ultimate Bravery. Use the optional same keyword to randomize the same users.
!r aram [same] Generates random champions and builds for ARAM. Use the optional same keyword to randomize the same users.
!r reroll Rerolls your current champion and build for Summoner's Rift/ARAM.
